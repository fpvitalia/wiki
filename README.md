# FPV Italia 🇮🇹

Il punto di riferimento italiano per il mondo dei droni FPV.

Guide, recensioni, normativa e community, tutto in un unico posto, costruito dalla community.

## 🚀 Quick Start

```bash
# Clona il repo
git clone https://github.com/fpvitalia/wiki.git
cd wiki

# Installa le dipendenze
npm install

# Avvia il server di sviluppo
npm run dev
```

Il sito sarà disponibile su `http://localhost:4321`.

## 📝 Contribuire

Vuoi scrivere una guida? Leggi [CONTRIBUTING.md](CONTRIBUTING.md) per tutti i dettagli.

**Modo veloce**: apri la pagina sul sito e clicca **"Modifica questa pagina"**, oppure crea un nuovo file in `src/content/docs/<categoria>/` direttamente da GitHub. Al salvataggio parte una Pull Request che il team revisiona e pubblica. Non serve installare nulla.
