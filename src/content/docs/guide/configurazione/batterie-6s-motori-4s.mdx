Con Betaflight 4.0 e successivi è possibile usare batterie **6S** su motori nati
per **4S**, limitando via software la potenza erogata. È un modo per sfruttare i
vantaggi del 6S (meno corrente a parità di potenza, quindi meno calore) su un
setup pensato per il 4S.

<Aside type="caution" title="Prerequisito fondamentale">
Qualsiasi componente collegato **direttamente** alla batteria deve reggere oltre
**25 V** (una 6S a piena carica arriva a 25,2 V). Verifica ESC, VTX, camera e ogni
regolatore alimentato dal pacco: se uno solo non regge il 6S, **non procedere**.
</Aside>

## 1. Calcola il Motor Output Limit

Si usa la funzione `motor_output_limit` per "trasformare" i KV dei motori. La
formula è:

```
(kv_desiderati * 100) / kv_attuali = motor_output_limit
```

Esempio: per far comportare dei motori da 2400 KV come dei 1750 KV →
`(1750 * 100) / 2400 = ~73`. Imposterai quindi un limite del 73%.

![Calcolo del motor output limit](/images/guide/configurazione/batterie-6s-motori-4s/kv-converter.png)

## 2. Alza il Max Cell Voltage

Nel tab **Power & Battery** di Betaflight, imposta **Maximum Cell Voltage** a
**4.4 V**, così il software distingue correttamente una 6S da una 4S al momento del
rilevamento del numero celle.

![Impostazione Maximum Cell Voltage](/images/guide/configurazione/batterie-6s-motori-4s/max-cell-voltage.png)

## 3. Prepara i profili PID

Copia il profilo PID che usi sugli altri due profili disponibili, con il pulsante
**Copy Profile Values**: useremo tre profili per gestire automaticamente 4S e 6S.

![Copia dei profili PID](/images/guide/configurazione/batterie-6s-motori-4s/copia-profili-1.png)

![Selezione del profilo di destinazione](/images/guide/configurazione/batterie-6s-motori-4s/copia-profili-2.png)

## 4. Configura i profili dalla CLI

L'idea: Betaflight cambia profilo **in automatico** in base al numero di celle
rilevato. Nella [CLI](/guide/configurazione/guida-cli-betaflight/) configura:

<Steps>

1. **Profilo 0** (per le 4S):
   ```
   profile 0
   set auto_profile_cell_count = 4
   ```

2. **Profilo 1** (per le 6S, con il limite calcolato al punto 1):
   ```
   profile 1
   set auto_profile_cell_count = 6
   set motor_output_limit = 73
   ```

3. **Profilo 2** (fallback automatico per qualsiasi altro caso):
   ```
   profile 2
   set auto_profile_cell_count = -1
   ```

4. Salva:
   ```
   save
   ```

</Steps>

## 5. Verifica

Nel tab **PID Tuning**, collegando una 6S il profilo deve passare automaticamente
al profilo 1 (limitato), mentre con una 4S deve tornare al profilo 0. Se lo switch
avviene, la configurazione funziona.

<Aside type="caution" title="Turtle mode">
In passato la funzione **flip over after crash** (turtle mode) aggirava il
`motor_output_limit`, dando piena potenza ai motori. Il bug è stato risolto, ma
verifica di essere su una versione recente di Betaflight prima di affidarti al
turtle mode con una 6S su motori 4S.
</Aside>
